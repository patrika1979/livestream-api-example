# Kaiko livestream API - NodeJS example


## Requirements
* Kaiko API key [contact us](https://www.kaiko.com/pages/contact-1)
* NodeJS 10+
* npm 6+

## Getting started
```bash
$ npm install
$ export KAIKO_API_KEY=__REPLACE_WITH_YOUR_KEY__
$ node example.js
```
